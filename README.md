# Project Title

QR Code Generation

## Description

* QR Code Generation
* pyqrcode library used
* png library used

## Getting Started

### Dependencies

* Windows 7

### Installing

* pip install pyqrcode
* pip install pypng

### Executing program

* Copy "qrvalue.ini" in the tool path and edit the qr value as you want.
* Run the program
* The "qrcode.svg" and "qrcode.png" file will created in the same tool path
